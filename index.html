FAQ

这只是静态页面，可以直接部署吗？

可以。本页面仅依赖浏览器与 CDN，无需打包构建，适合 GitHub Pages。

如何迁移到自定义域名？

在仓库根目录添加 CNAME 文件写入你的域名，并在域名 DNS 设置 CNAME 到 LiveSearchbench.github.io 即可。
